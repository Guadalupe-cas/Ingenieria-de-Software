Modelos de contexto

¿Que es?

Diagrama de contexto de sistema representa todas las entidades externas que podrían interactuar con un sistema. Dicho diagrama retrata el sistema en el centro, sin detalles de su estructura interna, rodeado por todos los sistemas, ambientes y actividades con las que interactúa. El objetivo del diagrama es enfocar la atención en los factores externos y eventos que deberían considerarse en el desarrollo de un set completo de requerimientos y restricciones del sistema.

Concepto Importante

Los diagramas de contexto pueden utilizar diferentes tipos de representaciones como: Imágenes, óvalos figuras etc., cualquier tipo de representación que tenga un significado. Las personas, organizaciones, sistemas o procesos con los que se comunica el sistema son conocidos como entidades. Trabajar con los participantes del sistema para determinar cuál funcionalidad se incluirá en el sistema y cuál la ofrece el entorno del sistema.

Caracteristicas:

  • Debe realizarse durante el proceso para limitar los costos del sistema.
  • Este modelo sirve para representar los limites del sistema.
  • Este diagrama es conocido como el nivel 0 del diagrama de flujo.
